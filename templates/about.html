Application Core Stack

This platform is engineered as a decoupled secure enterprise SaaS system, featuring distinct functional layers:

  • Presentation Layer: HTML5, Space Grotesk typography, glassmorphism CSS custom properties, and real-time interactive ApexCharts dashboards.
  • Application Controller: Managed by Python Flask server executing cryptographic password hashing with bcrypt and CSRF request validation.
  • Persistence Layer: SQLite backend mapping normalized users, scan histories, and quiz certificate tracking using SQLAlchemy ORM.
  • AI Classification Engine: Scikit-Learn's Random Forest classifier training 100 decision trees to execute binary threat predictions.

Machine Learning Parameters

When a URL is submitted, the custom feature extractor isolates key variables to build the classifier vector:

Variable Checked Threat Trigger Signature
url_length Flagged if overall length >= 54 characters.
having_ip_address IPv4/IPv6 address used as hostname instead of domain.
shortening_service Obfuscated using bit.ly, tinyurl, cutt.ly, etc.
prefix_suffix Hyphen character (-) in domain host (e.g. secure-paypal.com).
having_sub_domain Domain has >= 3 dot-separated parts.
ssl_final_state Scheme is unsecure HTTP instead of secure HTTPS.
